# CsoundUnity_MTEC340_DC

This repo was made for the Programming Indie Game Sound (MTEC-340-001) class at Berklee College of Music under the tutelage of David Cardona.

It contains a Unity project that utilises CsoundUnity to implement a dynamic weather system using a Cabbage audio generator instrument. Additionally, this repository also contains the documentation I wrote for the class as a quick "Get Started" guide.
